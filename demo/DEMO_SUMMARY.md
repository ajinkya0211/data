## Troubleshooting During Demo

- **If backend fails**: Check virtual environment activation, restart with `source venv/bin/activate && python main.py`
- **If frontend fails**: Check port 3000, restart with `npm run dev`
- **If execution fails**: Check Python dependencies, ensure pandas/matplotlib available
- **If AI is slow**: The 2-second delay is intentional for demo realism
- **System Status**: Use the System Status button to monitor all services
